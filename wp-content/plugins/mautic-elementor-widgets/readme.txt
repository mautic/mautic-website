== Description ==
Adds custom Elementor widgets for Mautic, including a responsive Mobile Menu and a dynamic Services grid.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/mautic-elementor-widgets` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use the widgets in the Elementor editor.
